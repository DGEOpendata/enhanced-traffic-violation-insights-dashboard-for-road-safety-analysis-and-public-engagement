# Enhanced Traffic Violation Insights Dashboard

## Overview
This project demonstrates how to create an interactive dashboard for analyzing traffic violations using the 'Traffic Fines Data (2022-2025)' dataset. The dashboard includes visualizations such as heatmaps, trend lines, and bar charts, offering insights into traffic violation patterns.

## Prerequisites
1. Python 3.x
2. Required Python libraries:
   - pandas
   - matplotlib
   - seaborn

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/your-repo/traffic-violation-dashboard.git
   
2. Navigate to the project directory:
   bash
   cd traffic-violation-dashboard
   
3. Install dependencies:
   bash
   pip install -r requirements.txt
   

## Dataset
Ensure the dataset file `Traffic_Fines_2022-2025.csv` is placed in the project root directory.

## File Structure

.
├── Traffic_Fines_2022-2025.csv
├── dashboard.py
├── README.md
└── requirements.txt


## Running the Dashboard
1. Run the Python script:
   bash
   python dashboard.py
   
2. Visualizations will be displayed inline or saved as image files, depending on the setup.

## Features
- **Heatmap**: Displays the distribution of ticket types over the years.
- **Trend Line**: Analyzes the trend of total tickets issued annually.
- **Bar Chart**: Highlights the top 10 most frequent traffic violations.
